Це тестовий файл, створений для демонстрації роботи з Git та GitHub у межах навчального проєкту з DevOps CI/CD. Файл містить довільний текст, який можна використовувати для експериментів з різними командами Git, включаючи створення комітів, відстеження змін та роботу з гілками. Цей простий приклад допоможе краще зрозуміти основи версійного контролю.
